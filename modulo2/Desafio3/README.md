## Desafio Covid-19



```
Objetivo: 

Neste projeto, iremos investigar os dados da disseminação do COVID-19 no Brasil desde o início da pandemia em fevereiro de 2020. Iremos estudar os padrões de disseminação da doença, investigar os números de infectados, de recuperados e de óbitos, para construir modelos com Python e Machine Learning para prever os números nos próximos dias e o ponto de virada da curva de infecção baseado em alguns cenários.
```
